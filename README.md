# AndroidLog
AndroidLog - Simple Android Logs

_Usage_ 
-----------------
* ```ALog.d("Message");```
* ```ALog.d("Message1","Message2","Message3");```
* ```ALog.d(1,2,3,4);```
* ```ALog.setHierarchy(2).setTag("ALOG");```
* ```ALog.e("Message");```
* ```ALog.showHierarchy(false).showHeaderAndFooter(false).d("Hello World");```
